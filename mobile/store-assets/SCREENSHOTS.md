# Screenshots Guide — BrindaWorld Mobile

## Required Sizes
### iOS
- 6.7" (1290x2796) — iPhone 14 Pro Max
- 6.1" (1170x2532) — iPhone 14
- 5-10 screenshots per size

### Android
- Phone (1080x1920) — standard
- 7" Tablet (1600x2560) — optional

## Screenshot Content (with demo data)
1. **Home screen** — Hero with "She Can Be Anything" and sign-in buttons
2. **She Can Be grid** — 12 profession cards (Judge, Astronaut, Engineer, etc.)
3. **Chess game** — Board in progress (use web screenshot)
4. **Quiz in progress** — Astronaut question with options
5. **Dashboard** — Activity stats (sessions, minutes, games)
6. **French mode** — Same screens with FR language toggle active

## How to Capture
1. Run `npx expo start` in mobile/
2. Open on device or simulator
3. Sign in with demo account
4. Navigate to each screen
5. Use device screenshot (Power + Volume Down)

## Notes
- Use demo mode data for consistent screenshots
- Ensure no real user data is visible
- Add device frames using https://mockuphone.com
