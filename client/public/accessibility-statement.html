Accessibility Statement

BrindaWorld is committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience for everyone and apply the relevant accessibility standards.

Conformance Status

BrindaWorld aims to conform to WCAG 2.1 Level AA. We have implemented:

  • Skip navigation link for keyboard users
  • ARIA labels on interactive elements
  • Sufficient color contrast (4.5:1 minimum)
  • Reduced motion support for vestibular disorders
  • Form labels and error announcements
  • Language attribute on document root
  • Focus management for modal dialogs

Known Limitations

The interactive chess board (Chessground) has limited screen reader support. We are working with the library maintainers to improve this.

Feedback

If you encounter accessibility barriers, please contact us at hello@brindaworld.ca.
